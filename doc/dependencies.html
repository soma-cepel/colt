Dependencies, Versions and their Homepages
Colt contains all dependencies necessary to run the software "out of the box" with zero installation and configuration effort. A single download and untar step will get you operational. This gets you up and running in 5 minutes with the least amount of hassle, on any box.

Developers and advanced users want to know the dependency set, and tinker with it in interesting ways. The following dependencies exist (left component depends on right component):

Product Dependency and Homepage Javadoc API License
colt javadoc Apache style
jdk-1.2.x or higher (we test with the sun-1.4.2 jdk on Linux) javadoc n/a
concurrent-1.3.2 javadoc Public Domain
corejava javadoc LGPL
com.imsl.math javadoc Apache Style

Chances are good that colt will also run with earlier or later versions of the above components - just drop them in. We can not guarantee that such other versions will work, of course. We are interested in feedback as to what does or does not work for you.

Before you submit bug reports, please check if the bug is reproducible with the versions listed above, and let us know.
